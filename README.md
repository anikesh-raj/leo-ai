# ⚡ Overview

LEO is not a chatbot.

It is a **local-first AI Operating System layer** that executes real-world actions across your system, applications, and devices.

> Speak your command. LEO executes it.

Built on **Electron + Vite + React**, powered by **Google Gemini Live API** and **Groq**, LEO bridges the gap between human voice and OS-level execution. It runs on your machine, stores nothing in the cloud, and gives your AI real hands.

# 🏗️ Architecture

```
┌─────────────────────────────────────────────────────────┐
│                    LEO AI — Electron App                 │
│                                                          │
│  ┌─────────────────┐    IPC Bridge    ┌───────────────┐ │
│  │  React Renderer │ <─────────────── │  Node.js Main │ │
│  │  (UI + Voice)   │ ──────────────── │  (OS Control) │ │
│  └─────────────────┘                  └───────────────┘ │
│         │                                    │           │
│  Gemini Live API                      File System        │
│  Groq SDK                             ADB Bridge         │
│  LanceDB (local)                      Nut.js Automation  │
└─────────────────────────────────────────────────────────┘
```

### Renderer Process (React)
- Handles UI, voice input, Gemini WebSocket stream
- Dispatches tool calls via `window.electron.ipcRenderer`

### Main Process (Node.js)
- Full OS access — files, apps, terminal, ADB
- Registers all IPC handlers

### IPC Bridge
```js
window.electron.ipcRenderer.invoke("tool-name", payload)
```

# 🚀 Installation & Setup

### Option A — Download Release (Recommended)

1. Go to [**Releases**](https://github.com/anikesh-raj/leo-ai/releases/latest)
2. Download `leo-ai-x.x.x-setup.exe`
3. Run the installer — follow setup wizard
4. Open **LEO AI** from Desktop shortcut
5. Go to **Settings → Command Center Vault**
6. Add your Gemini API key
7. Click **ACTIVATE** — LEO is online

---

### Option B — Build from Source

```bash
# 1. Clone
git clone https://github.com/anikesh-raj/leo-ai.git
cd leo-ai

# 2. Install
npm install

# 3. Setup env
cp .env.example .env
# Fill in your API keys

# 4. Dev mode
npm run dev

# 5. Build EXE
npm run build:win
```

# 🧩 Extending LEO

Add a new tool in 3 steps:

**1. Create `src/main/logic/your-tool.ts`**
```ts
import { ipcMain } from 'electron'
export default function registerYourTool() {
  ipcMain.handle('your-tool', async (_, args) => {
    // OS-level code here
    return 'result'
  })
}
```

**2. Register in `src/main/index.ts`**
```ts
import registerYourTool from './logic/your-tool'
registerYourTool()
```

**3. Add declaration + handler in `Iris-voice-ai.ts`**
```ts
// In functionDeclarations:
{ name: 'your_tool', description: 'When to call this', parameters: {...} }

// In onmessage handler:
} else if (call.name === 'your_tool') {
  result = await window.electron.ipcRenderer.invoke('your-tool', call.args)
```
